# **Personal Finance Checker 🏦💰**


Track your expenses, set budgets, and gain insights into your spending habits with this simple yet powerful personal finance tool. Perfect for anyone looking to take control of their finances! 📈

**✨ Features**
Expense Tracking - Log income and expenses with categories

Budget Planner - Set monthly budgets and get alerts

Visual Reports - Pie charts and graphs for spending analysis

Data Export - Export to CSV/PDF for records

Simple CLI Interface - Easy to use from terminal

Local Storage - Your data stays private on your machine

**🚀 Quick Start**
```
# Clone the repo
git clone https://github.com/yourusername/personal-finance-checker.git
cd personal-finance-checker

# Install dependencies
pip install -r requirements.txt

# Run the app
python finance_checker.py
```
Follow the interactive prompts to start tracking your finances!
